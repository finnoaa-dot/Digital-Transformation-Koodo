# Digital Transformation Elevates Customer Experience

A comprehensive case study on transforming customer support for Koodo, a leading prepaid mobile provider, through strategic research, design, and engineering.

## Project Overview

This project demonstrates how digital transformation can simultaneously improve customer experience and reduce operational costs. By leveraging user research and design thinking, we developed innovative solutions that streamlined support operations and enhanced user satisfaction.

![Project Overview](./assets/images/mockups/project-overview.png)

## Methodology

- **Competitive Analysis** – Benchmarking against industry leaders
- **Opt-In Panel Survey** – Quantitative insights from target audience
- **Moderated In-Depth Interviews** – Qualitative user discovery
- **Feature Desirability Survey** – Validation of proposed features
- **Experience Audit** – Comprehensive evaluation of current state
